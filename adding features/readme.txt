Реализация второй части проекта по Javascript "Практика построения объектов - прыгающие шары" с добавлением продвинутых функций.

Подробное условие смотри в файле "Добавление функций в нашу демонстрацию отбойных шаров MDN.pdf"

По сравнению с исходным условием отключена функция изменения цвета шаров при столкновении - при большом количестве шаров она вызывала дискомфорт для глаз.
